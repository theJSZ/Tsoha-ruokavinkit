# Tsoha-ruokavinkit
Sovellus sisältää tietoja ruokalajeista joita voi laittaa kotona.
Sovellus on tarkoitettu perheen yhteiseen käyttöön. Alustavasti olen ajatellut että kaikki käyttäjät
kuuluisivat samaan perheeseen mutta jos onnistun tekemään systeemin joka skaalautuisi useammalle perheelle
niin sitä parempi.
Jokaisesta ruuasta on tietokannassa nimi, valmistusaika, lista aineksista, ohje, ja mahdollisesti kuva.
Ruokalajeilla on myös avainsanoja esim "pääruoka", "keitto", "makea" jne.
Jokaisesta ruokalajista tallentuu myös tieto milloin sitä on valmistettu.
Ylläpitäjä voi lisätä tietokantaan ruokia ja muokata niiden tietoja.
Käyttäjä voi etsiä ruokalajeja niiden tietojen perusteella, arvostella niitä jollain asteikolla,
ja esittää ruokatoiveen joka näkyy muille perheenjäsenille.
Ruokatoive on voimassa kunnes ruoka merkataan tehdyksi tai käyttäjä vaihtaa toivettaan.
Käyttäjä voi kommentoida ruokia, mahdollisesti myös päivämäärän mukaan jos
jonain tiettynä kertana oli jotain erityistä mainittavaa.
